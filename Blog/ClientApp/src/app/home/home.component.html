About

I'm Sam Wheat. I live in Medford, Oregon. I am a software engineer. I write enterprise database applications using Microsoft .Net.

I write web applications and desktop applications. For the desktop I use Windows Presentation Foundation (WPF). For web applications I use a variety of tools and frameworks, the most predominant being Angular, RxJs, and TypeScript. I use ASP.Net core MVC and Razor pages as well.

Every application I have worked on in my career has been a database application. Because of this I've acquired a fair amount of experience using Microsoft SQL Server and to a lesser extent Oracle and MySQL. Entity Framework Core is a related technology that I have grown to appreciate and I use it everywhere I can.

I moved from San Diego to Medford in October 2022. I spent most of my career working in Los Angeles, Orange County, and San Diego as an independent contractor. Much of my work has been in finance and banking however I have had clients in manufacturing, broadcasting, medical, and construction services.

I spend a lot of time working on personal projects. This gives me an opportunity to try new tools and technologies as they become available and it helps me stay current in this fast moving industry. For the last eight years I've been working on a platform for working with economic data that is published by the St. Louis Federal Reserve Bank. Along the way I've written several utilities mostly related to the application service layer. I've spent quite a bit of time with Blazor - I even wrote an implementation of my Pivot Grid control for it. More recently I've been working on some controls using WinUI 3 and the Uno Platform.

When I'm not writing code I like to read up on economics and investing. I also enjoy hiking and mountain biking and working on home projects in my garage.
